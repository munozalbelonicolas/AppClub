# Smoke Tests

*Este checklist debe ejecutarse en menos de 10 minutos cada vez que se genera un nuevo APK/IPA.*

- [ ] **ST-01:** La aplicación se abre sin crashear en la pantalla de Splash.
- [ ] **ST-02:** El login con credenciales válidas funciona.
- [ ] **ST-03:** El feed de Novedades (Inicio) carga datos.
- [ ] **ST-04:** El menú de navegación (NavigationBar) responde.
- [ ] **ST-05:** Se puede visualizar el detalle de un Producto en la Tienda.
- [ ] **ST-06:** Se puede cerrar sesión correctamente.
